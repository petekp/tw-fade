Tailwind v4 Plugin

tw-fade

Elegant, CSS-driven scroll-edge masking. Add a single class to any scroll container; its content dissolves into the surface at the edges. Zero JavaScript.

Chrome Full scroll-gated fades in Chrome 120+ on desktop and Android. Edge Full scroll-gated fades in Edge 120+ on desktop and Android. Safari Full scroll-gated fades in Safari 26+ on macOS. Mobile Safari Full scroll-gated fades in Safari 26+ on iOS and iPadOS. Firefox Partial support in stable Firefox: the mask renders, but scroll-gating falls back to a static always-on fade. Firefox Nightly Full scroll-gated fades in Firefox Nightly, where scroll-driven animations are enabled by default. Opera Full scroll-gated fades in Opera 106+.

Bottom emphasis

fade-y fade-size-sm fade-size-b-lg

Top keeps the short shared size.

Bottom gets a longer fade for stronger affordance.

Use it for drawers, sheets, and scrollable previews.

Arbitrary values work too: fade-size-b-[6rem].

The override is per edge, not class-order dependent.
